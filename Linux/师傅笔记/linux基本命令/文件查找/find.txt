find,查找文件命令。

常用参数：

一、根据类型查找

-type

筛选条件有:
 f----普通文件
 d----目录文件

例如：查找根目录下的文件
# find / -type f

二、根据名称查找

-name----区分大小写
-iname---不区分字母大小写

例如：查找"/home"下所有txt档文件
# find /home -name "*.txt"

注：这里可能会有人感到疑惑，"txt档"应该算是类型查找才合理，但是不知通过类型查找又是怎样操作呢。因为在linux系统里文件名的后辍是没有意义的，linux并不注重文件名的后辍，这和windows系统是有很大区别的，所以把"txt"只看作是文件名称的一部分。

三、将查询结果传递给下一个命令

-exec

参数后面跟的是命令，它的终止是以";"为结束标志的，所以这句命令后面的分号是不可缺少的，考虑到各个系统中分号会有不同的意义，所以前面加反斜杠"\"。

例子：
# find /home/rise/ -name "*.txt" -exec iconv -f gbk -t utf8 {} -o /home/{} \;

该例子的意思是：查找出目录"/home/rise/"及其子目录下的所有的"txt"档文件，然后将它由原本是gbk码转换成utf8码，并放到"/home/home/rise"目录下

说明：上面语句的"{}"符号代表的是查询的结果，即"/home/rise/"目录下所有的"txt"档文件。


这里顺便讲解一下上面的那个例子，该例子需要一个前提，"/home/rise/"下的目录及子目录在"/home/home/rise"下必须全都有，所以得先创建目录，操作如下：

# find /home/rise/ -type d -exec mkdir -p /home/{} \;
